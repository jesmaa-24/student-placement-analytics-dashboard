# student-placement-analytics-dashboard
📊 Power BI dashboard analyzing student placement trends, placement rates, salary packages, CGPA impact, internships, and project-based employability insights.
Project Overview
This project presents an interactive Power BI dashboard that analyzes student placement data to identify factors influencing placement success. The dashboard provides insights into placement rates, salary packages, academic performance, internships, and project experience.
Features
Placement Status Analysis by Branch
Placement Rate by Branch and College Tier
Overall Placement Distribution
Projects & Internships Impact on Placement
CGPA vs Placement Rate and Average Salary
Key Performance Indicators (KPIs)
KPIs
Total Students
Placed Students
Placement Rate (%)
Average Package (LPA)
Tools Used
Microsoft Power BI
DAX
Data Visualization
Data Analysis
Key Insights
Higher CGPA is associated with improved placement rates and salary packages.
Students with more projects and internships tend to achieve better placement outcomes.
Placement performance varies slightly across branches and college tiers.
