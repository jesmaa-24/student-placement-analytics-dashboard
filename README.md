# 🎓 Student Placement Analytics Dashboard

## 📌 Project Overview

The Student Placement Analytics Dashboard is an interactive Power BI project designed to analyze student placement trends and identify factors that influence placement success. The dashboard provides insights into placement rates, salary packages, academic performance, internships, projects, and college tiers.

---

## 📊 Dashboard Features

- Placement Status Analysis by Branch
- Placement Rate by Branch and College Tier
- Overall Placement Distribution
- Projects & Internships Impact on Placement Rate
- CGPA vs Placement Rate and Average Salary Analysis
- Interactive KPI Cards for quick insights

---

## 🎯 Key Performance Indicators (KPIs)

- Total Students
- Placed Students
- Placement Rate (%)
- Average Package (LPA)

---

## 🛠️ Tools & Technologies Used

- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Business Intelligence
- Data Analytics

---

## 📈 Dashboard Insights

- Higher CGPA is associated with better placement outcomes and salary packages.
- Students with more projects and internships generally achieve higher placement rates.
- Placement rates vary across different branches and college tiers.
- CSE and IT branches contribute the highest number of placed students.
- Academic and practical experience play a significant role in employability.
